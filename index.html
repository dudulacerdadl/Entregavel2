Robots



Os Robots são programas automatizados que buscam documentos em páginas Web, a fim de indexá-los. Para controlar esses documentos durante a busca, é criado um arquivo chamado robots.txt, que é usado como filtro para os motores de busca, permitindo ou bloqueando o acesso a partes determinadas de um site. Esses arquivos são colocados no diretório raiz do endereço web.


O Google, porém, adverte que esse recurso não funciona como um mecanismo para manter uma página da Web fora dos resultados da pesquisa do Google. Para fazer isso, bloqueie a indexação com NOINDEX ou proteja a página com uma senha.


O parâmetro NOINDEX diz ao robô que o conteúdo daquele site não deve ser indexado e NOFOLLOW indica que os possíveis links ali existentes não devem ser analisados.


Ao visitar um site, os robôs buscam primeiro pelo arquivo robots.txt. Se o conteúdo deste arquivo listar algo conforme segue:

                User-agent: *
                Disallow: /

então o robô saberá que deve deixar o local sem buscar por nada ali. Outra maneira de se conseguir algo semelhante é através de Meta Tags, colocadas estrategicamente nos cabeçalhos de páginas HTML:


<META NAME="ROBOTS" CONTENT="NOINDEX,NOFOLLOW">

O Google disponibiliza em sua central de desenvolvedores, uma documentação ampla para criação de robots, podendo ser acessada pelo link: Criar e enviar um arquivo robots.txt


Mesmo criando corretamente um robot, ainda há algumas limitações que podem fazer com que a criação dele não seja o ideal para sua página web, tento de analisar se essa seria realmente a melhor opção. Algumas dessas limitações são que:


  • As diretivas do arquivo robots.txt podem não ser compatíveis com todos os mecanismos de pesquisa
  • Rastreadores diferentes interpretam a sintaxe de maneiras distintas
  • Uma página não permitida no arquivo robots.txt ainda poderá ser indexada se estiver vinculada a outros sites

Se o robô não encontrar o arquivo robots.txt e não houver nenhuma Meta Tag que o proíba formalmente de analisar um site, a decisão sobre o que fazer passa a ser do próprio robô. Dependendo do site de busca ao qual ele pertence, ele poderá ou não prosseguir com a busca. A maioria optará por prosseguir.
