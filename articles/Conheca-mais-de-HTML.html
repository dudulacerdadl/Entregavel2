Robots

A internet começou com o objetivo de comunicação militar e como padrão de publicação e distribuição de textos científicos e acadêmicos.

O objetivo da internet era se focar nos hipertextos, onde o leitor escolheria o que ele gostaria de ler.

Para o acesso a internet, deve haver uma comunicação Cliente Servidor, no qual o cliente gostaria de realizar o acesso a um caminho, ele faz um pedido ao servidor, que envia uma respostas em formato de HTML.

Para que todo esse esquema funcione, precisamos de 3 elementos:

  • URL: Um localizador de recursos
  • HTTP: Um protocolo de transferência de Hipertexto
  • HTML: Uma linguagem de marcação de Hipertexto

O W3C veio para implementar e definir padrões para facilitar a utilização de vários navegadores e facilitar a programação.

O HTML é usado para realizar a marcação das páginas Web, o CSS usado para estilização e o JS responsável pela inteligência da página.

O arquivo HTML nada mais é do que um arquivo de texto, com marcações HTML que dão sentido ao conteúdo.

Uma página é um arquivo HTML e seus complementos, enquanto um site, é uma pasta de vários arquivos HTML e seus complementos.

É sempre importante ter as tag aninhadas, ou seja, seguir a hierarquia lógica, com a primeira que abre, sendo a última que fecha.

Estrutura Semântica

Todo conteúdo de uma página HTML tem que estar dentro da tag <html>

Dentro da tag <html> existem duas tags principais:

  1. <head>, que é usada para definir informações que não estão visíveis no corpo da página, mas que são úteis para a mesma, com título e linguagem. Também são usadas para linkar outros arquivos que serão usados pela página, como CSSs e JSs
  2. <body>, que é usado para escrever todo o corpo da página web

Os comentários no HTML são escritos com <!-- … -->.
