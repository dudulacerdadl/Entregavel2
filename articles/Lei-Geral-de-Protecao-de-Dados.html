Lei Geral de Proteção de Dados (LGPD)

Em 2015 houve uma aprovação de lei europeia que aplicava punições severas para empresas que compartilhassem indevidamente os dados dos cidadãos europeus com países sem proteção de dados.

Pouco mais de 6 meses depois, o projeto de lei brasileira foi aprovado, com praticamente uma cópia da lei europeia.

Hoje as empresas têm uma capacidade de manipulação de dados tão grande, que pode chegar a influenciar uma pessoa a comprar coisas que ela nem sabia que precisava.

A lei de proteção de dados vem tanto na questão mercadológica, como também em uma proteção da nossa privacidade online.

A informação é propriedade da pessoa, a empresa apenas tem o direito, dependendo do nível de aprovação, para manipular esses dados.

Hoje, com a LGPD, a manipulação indevida de dados pessoais é passível de punição, coisas que não eram anteriormente.

Os dados pessoais se referem a dados que sejam utilizados para identificar um indivíduo. Podem ser adicionados a essa lista, dados pessoais sensíveis, como dados de orientação sexual, religioso, entre outros, que podem causar qualquer tipo de preconceito para pessoa.

Uma empresa controladora é quem coleta e controla os dados necessários.

Uma empresa operadora é quem trata o dado passado por uma empresa controladora conforme necessário.

Deve ser estabelecido muito bem o contrato entre essas duas empresas.

Um incidente é qualquer ação, proposital ou acidental que possa permitir o acesso não autorizado ou causar a distribuição, perda, alteração, comunicação, ou qualquer outra forma de tratamento inadequado ou ilícito de dados pessoais, que cause a queda de confidencialidade, de integridade ou de disponibilidade de informação.
